## 递推方程
可以得到如下的递推方程：
```
dp[i, w] = max(dp[i-1, w], dp[i-1, w-wi] + vi), if w >= wi
dp[i, w] = dp[i-1, w], else
```
同时，在遍历W的时候，可以从后向前遍历，可以使用一维数组实现。

## （可能的）改进
由于第二题数据量较大，而且通过查看数据，发现W相对wi来说较小，似乎很容易就会装不下。所以先对物品按照weight排序，weight较小的在后面，先遍历。这样，当w < wi的时候，说明继续遍历下去仍然是小于的，没有必要继续遍历。
